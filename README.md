# 🛒 Retail Sales & Customer Segmentation Analytics

## 📌 Objective
Analyze 2+ years of retail sales data to identify revenue trends, 
top-performing products, regional performance, and segment customers 
using RFM analysis to improve business strategy.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI
- Excel
- Git & GitHub

## 📊 Key Findings
- Overall profit margin: 12.5%
- West region is the highest revenue region ($725K)
- Loss-making sub-categories: Bookcases, Supplies, Tables
- High discounts (>40%) cause average loss of $106 per order
- 189 Champion customers drive premium revenue
- 189 Lost customers need re-engagement campaigns

## 🔍 Project Steps
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. SQL Analysis
4. RFM Customer Segmentation
5. Power BI Dashboard
6. Business Insights & Recommendations

## 💡 Business Recommendations
1. Reduce discounts above 40% — causing direct losses
2. Re-engage 189 lost customers with targeted campaigns
3. Focus marketing on West & East regions (highest revenue)
4. Discontinue or reprice Bookcases & Tables (loss-making)
